# create-web3js-app

> create-web3js-app command-line tool.

## Install

```
$ npm install --global create-web3js-app
```

## Usage

```
$ create-web3js-app --help

  Usage
    $ create-web3js-app [input]

  Options
    --postfix  Lorem ipsum  [Default: rainbows]

  Examples
    $ cli-name
    unicorns & rainbows
    $ cli-name ponies
    ponies & rainbows
```
