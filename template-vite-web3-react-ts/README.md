# Vite-web3-react-ts

## Requirements

- NodeJS
- Yarn ($ npm install yarn --global)

## Development

```
$ cd backend
$ yarn
$ yarn start    (open http://localhost:3030)

$ cd ui
$ yarn
$ yarn dev      (open http://localhost:3000)
```

## Build and Deploy

```
$ cd backend
$ yarn build

$ cd ui
$ yarn build
```

Deploy using Vercel:
```
$ cd <this-project>
$ npx vercel
```
